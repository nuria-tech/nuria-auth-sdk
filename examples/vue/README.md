# Vue Example

Minimal Vue integration using `@nuria-tech/auth-sdk/vue`.

## Files

- `src/auth.ts`: SDK client instance
- `src/composables/usePageAuth.ts`: auth composable
- `src/components/AuthStatus.vue`: auth UI
- `src/views/CallbackView.vue`: callback handler view

## Install

```bash
npm install @nuria-tech/auth-sdk vue
```
