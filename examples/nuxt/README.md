# Nuxt Example

Minimal Nuxt integration with `@nuria-tech/auth-sdk/nuxt` + `@nuria-tech/auth-sdk/vue`.

## Files

- `composables/useAuthClient.ts`: Nuxt auth client factory
- `composables/usePageAuth.ts`: session composable
- `pages/index.vue`: auth status page
- `pages/callback.vue`: callback handler page

## Install

```bash
npm install @nuria-tech/auth-sdk nuxt vue
```
